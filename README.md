# ChessIt
ChessIt is a powerfull chess analysis application built using the language of Python. It uses the python-chess library among with PyQt6 to make a powerfull combination with GUI and chess. It also integrates the latest Stockfish edition.

## Requirements
The following shows the libraries along with external requirements for this application to run error-free and smoothly.

### Libraries
- python-chess
- PyQt6
- stockfish

### External
- Stockfish 18 (https://stockfishchess.org/download/)
- 64-bit Windows / Mac (only for source code) Computer

### Pro Tip
Type in the command `pip install PyQt6 python-chess stockfish` to directly install all the required libraries all at once!
